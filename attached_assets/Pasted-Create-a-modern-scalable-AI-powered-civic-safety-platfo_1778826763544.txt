Create a modern, scalable, AI-powered civic safety platform called **RoadSoS AI** — a smart road safety, emergency response, and infrastructure transparency ecosystem for citizens, emergency responders, and government authorities.

The platform should have a clean, futuristic UI with responsive dashboards for mobile and web. Build it as a production-ready application with modular architecture, authentication, real-time location services, AI integrations, and analytics.

Core idea: citizens can report road issues, accidents, and emergencies with photos/videos and GPS location, while authorities receive intelligent routing, analytics, and transparency dashboards.

Main modules and features to implement:

---

## 1. Authentication & User Management

Create a secure authentication system with:

* Google Sign-In
* Email & Password Login
* Mobile OTP Login
* Role-Based Authentication:

  * Citizen
  * Admin
  * Government Authority
  * Emergency Services

Include:

* JWT/session management
* User profile management
* Aadhaar verification flow for trusted users
* Verified badge system
* Secure encrypted storage

---

## 2. Smart Complaint Reporting System

Allow users to report:

* Potholes
* Road damage
* Accidents
* Water logging
* Signal failures
* Illegal parking
* Traffic congestion

Features:

* Upload images/videos
* Voice complaint support
* GPS auto-detection
* Google Maps integration
* Live location sharing
* Interactive road marking

Implement a ticketing workflow similar to Uber/Zomato support:

* Auto-generated Ticket ID
* Status flow:

  * Submitted
  * Under Review
  * Assigned
  * In Progress
  * Resolved
* Real-time notifications
* Complaint timeline tracking

---

## 3. Road Transparency Dashboard

For each road segment display:

* Road type:

  * NH
  * SH
  * MDR
  * Rural Road
* Last maintenance date
* Contractor details
* Tender details
* Department responsible
* Road quality score
* Estimated road lifespan

Visualize all information on maps and dashboards.

---

## 4. Smart Authority Routing & Escalation

Build AI-based complaint routing logic.

The system should automatically identify:

* Road ownership
* Complaint category
* Geographic jurisdiction
* Responsible department

Automatically forward complaints to:

* Executive Engineers
* Highway Departments
* Municipal Authorities
* Traffic Police

Add escalation logic:

* Auto reminders
* Escalation to higher authorities
* Public visibility score if delayed
